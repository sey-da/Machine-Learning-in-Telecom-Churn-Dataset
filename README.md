## 📌 Project Overview

This project develops a machine learning pipeline to predict customer churn in the telecommunications industry.

The objective is to identify high-risk customers using structured data and enable data-driven decision-making for customer retention strategies. The project covers the full workflow from data preprocessing to model evaluation and optimization.

---

## 🚀 Key Highlights

- End-to-end machine learning pipeline implementation  
- Data preprocessing, feature selection, and encoding  
- Comparative analysis of classification models (Logistic Regression vs Random Forest)  
- Model evaluation using multiple performance metrics (Accuracy, Precision, Recall, F1-score, ROC-AUC)  
- Hyperparameter tuning using GridSearchCV  
- Cross-validation with Stratified K-Fold  
- Threshold optimization based on Precision-Recall trade-off  
- Model interpretability through feature importance and coefficient analysis
